豆瓣电影封面
== Description ==

这是一款基于豆瓣API的获取电影信息生成封面卡片的wordpresss插件。

官方演示/Demo: 

[**豆瓣封面插件官方演示**](https://www.lylares.com/a-wordpress-plugin-of-douban-movie.html"豆瓣封面")如果您有任何问题或建议也可以留言，谢谢支持。

== Installation ==

1. 下载并解压压缩包到 '/wp-content/plugins/' 目录下
2. 后台-插件-启用插件
3. 文章编辑器-文本模式-「豆瓣封面」快捷按钮-id处填写要生成的豆瓣id

* 豆瓣id 来自于豆瓣链接如：https://movie.douban.com/subject/1292052/ 
则此 ID为：1292052
